# Title
A python script to automatically change Desktop wallpaper on **Linux** only

# Features
- It can download wallpaper
- Automatically change the wallpaper
- Save the wallpaper after setting up
- Can search your type of wallpaper by keywords
- Can download random wallpapers
- Quick and Easy

# Language
- Python

# Instruction
- Open the script
- Add your own wallhaven api key [you can get it by loging in wallhaven]
- Save and Run the script
- Enter your keyword or just hit 'Enter' 
- Wait for a few seconds
- Done. Your wallpaper is set
